Description:

This application processes images in Python with PySide6. To speed up some processing, functions have been written in Cython and others in C++ with PyBind11 (Linux version only).

The main functions are as follows:

  • Geometric transformations
    • Vertical symmetry
    • Horizontal symmetry
    • Rotation
    • Diagonalization
    • Medallion
    • Translation
    • Zoom +
    • Zoom -
    • Crop
  • Local filters
    • Brightness
    • Contrast
    • Luminance
    • Modifying RGB Colors
    • Red effect
    • Threshold
    • Binarization
    • Red, green and blue filters
    • Windowing
    • Greyscale
    • Shade of colors
    • False colors
    • Sepia
  • Neighborhood filters
    • Average
    • Median filter
    • Gaussian filter
    • Horizontal gradient
    • Vertical gradient
    • Edge detection
    • Erosion
    • Dilation
    • Gamma Filter
    • Dot transformation
    • Dash transformation
  • Noise Simulator
    • White noise
    • Salt and pepper noise
    • Gaussian noise
  • Encryption
    • Pixelation
    • Random Key Row Shift Encryption
    • Line shift encryption with fixed key
    • Decryption
    • Inserting a secret image
    • Inserting a secret text
  • Image overlay
    • Red, green, blue, white, black and tinted background
    • image overlay
    • Anaglyphs
    • Image overlay
  • Format change
    • Monochrome
    • Index8
    • ARGB32
    • Grayscale8
    • Image size reduction
  • Development of new functions
    • Color TV pattern
    • Black and white TV pattern
    • Hexadecimal code of the image
    • Histogram
    • Sandbox functions

Limitation:

This application only supports images:

  • in jpg or png format
  • modest size
  • 4-byte RGBA format
